HDFSFormatWriteV1.spl

streamsx.hdfs2.helpers -> com.ibm.streamsx.hdfs2.helpers.blog-> HDFSFormatWriteV1.spl

This namespace hosts intermediate versions of HDFSWriteFormat as it's built up to the final form. The blog post refers to these versions.

  • HDFSFormatWrite_V1 encapsulate the scan-read-write example, hard-coding the formatter to be the txt format.
  • HDFSFormatWrite_V2 Let the formatting operator be an argument.
  • HDFSFormatWrite_V3 The above, but now the formatting operator has a default, so that we can use it just as we used V1.

public composite HDFSFormatWrite_V1(input inStream)

Convert a tuple to a string and then write to HDFS. The operator converts input tuples on its input stream to strings, and then sends them to the HDFS2FileSink.

Parameters
